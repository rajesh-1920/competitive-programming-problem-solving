## How to Use

1. Open the folder for a platform.
2. Search for the problem name/code.
3. Compile and run the C++ solution locally.

Example compile command:

```bash
g++ -std=c++17 -O2 -Wall -Wextra -o solve problem.cpp
./solve
```
